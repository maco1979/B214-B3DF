# my-first-project

## 📖 项目简介
这是我的第一个项目，基于AI决策系统开发，实现了PTZ云台控制、摄像头连接、区块链集成等核心功能。

## 🛠️ 技术栈
- 前端：React、Vite、TypeScript
- 后端：FastAPI、Python
- 区块链：智能合约、贡献奖励系统
- 其他：PTZ云台控制、视觉识别、边缘计算

## 🚀 运行步骤
1. 克隆项目：git clone 你的仓库地址.git
2. 进入项目目录：cd my-first-project
3. 安装依赖：npm install 或 pip install
4. 启动项目：npm run dev
