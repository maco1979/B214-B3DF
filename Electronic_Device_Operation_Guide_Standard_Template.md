## 安全警示

⚠️ **警告**: 使用前请仔细阅读本说明书中的所有安全警示，以避免人身伤害或设备损坏。

### 1.1 电气安全

```
⚠️ 警告 - 电气安全
• 使用前请确保电源电压与设备要求一致
• 请勿用湿手操作设备或插拔电源
• 设备出现异常时，请立即断开电源
• 请勿自行拆解设备，避免触电危险
• 请使用原装或认证电源适配器
• 避免电源线受压或扭曲
• 定期检查电源线是否有损坏
```

### 1.2 机械安全

```
⚠️ 警告 - 机械安全
• 请勿将手指或其他物体伸入设备运动部件
• 移动设备时请关闭电源并小心操作
• 请勿在设备运行时进行清洁或维护
• 确保设备放置在稳固、平整的表面上
• 避免设备跌落或受到强烈撞击
```

### 1.3 热安全

```
⚠️ 警告 - 热安全
• 设备运行时表面可能发热，请勿触摸
• 请勿在设备上覆盖物品，影响散热
• 请勿在易燃物品附近使用本设备
• 长时间使用后，设备需要适当休息
• 确保通风口畅通无阻
```

## 快速入门

### 3.1 开箱检查步骤

📦 **开箱检查（5W2H方法）**
• **What**（检查什么）: 确认包装内容
• **Who**（谁来检查）: 用户或专业人员
• **When**（何时检查）: 收到产品后立即检查
• **Where**（在哪里检查）: 干净、平整的桌面
• **Why**（为什么检查）: 确保配件完整、无损坏
• **How**（如何检查）: 按包装清单逐一核对
• **How much**（检查程度）: 100%核对

1. 检查外包装是否完好无损
2. 核对产品型号与订单是否一致
3. 按清单核对所有配件是否齐全
4. 检查产品外观是否有明显损坏
5. 确认序列号与包装一致

**包装清单**:
- [ ] 主机 × 1
- [ ] 电源适配器 × 1
- [ ] 说明书 × 1
- [ ] 保修卡 × 1
- [ ] [其他配件] × [数量]

### 3.2 初始设置步骤

⚙️ **初始设置指南**

1. **电源连接**
   - 连接电源适配器到设备
   - 插入匹配的电源插座
   - 确认电源指示灯亮起

2. **系统初始化**
   - 首次开机可能需要2-3分钟
   - 按照屏幕提示选择语言和地区
   - 设置用户账户和密码

3. **网络连接（如适用）**
   - 选择可用的WiFi网络
   - 输入网络密码
   - 确认网络连接成功

### 3.3 首次使用

1. 确保设备已充满电（首次使用前建议充电[X]小时）
2. 按照3.2节完成初始设置
3. 阅读本说明书中的安全警示

## 基本操作

### 4.1 开机/关机操作

🔌 **开机操作（符合GB 4943.1-2022）**
1. 检查电源连接是否正常
2. 按下电源键（持续3秒）
3. 等待系统启动完成（指示灯稳定）
4. 确认显示屏正常显示

🔌 **关机操作（符合GB 4943.1-2022）**
1. 保存所有未保存的工作
2. 通过系统菜单选择"关机"
3. 等待设备完全停止运行（指示灯熄灭）
4. 断开电源连接（如长时间不使用）

### 4.2 基本功能使用

📋 **基本功能操作指南**

• **功能A操作**:
  1. 按下[功能A键]或在屏幕上选择[功能A]
  2. 根据需要调整参数设置
  3. 确认设置并开始使用

• **功能B操作**:
  1. 进入[设置]菜单
  2. 选择[功能B设置]
  3. 按照向导完成配置
  4. 保存设置并返回主界面

### 4.3 连接与配对

🔗 **连接操作指南**

• **WiFi连接**:
  1. 进入设置菜单
  2. 选择网络设置
  3. 选择可用网络
  4. 输入密码
  5. 确认连接

• **蓝牙配对**:
  1. 开启蓝牙功能
  2. 搜索可用设备
  3. 选择目标设备
  4. 输入配对码（如需要）
  5. 确认配对成功

## 维护与保养

### 6.1 日常清洁保养

🧽 **日常清洁指南**

⚠️ **安全警告**: 清洁前必须断开电源并等待设备完全冷却

**清洁工具**:
• 柔软的微纤维布
• 75%医用酒精或专用清洁剂
• 干燥的软毛刷（清洁缝隙）

**清洁步骤**:
1. 关闭设备并断开电源
2. 等待设备完全冷却（至少30分钟）
3. 用干燥软布清除表面灰尘
4. 轻微湿润布料，拧干后擦拭表面
5. 用干燥布料擦干残留水分
6. 清洁完成后等待完全干燥再开机

**清洁频率**:
• 外观表面: 每周1次
• 屏幕表面: 每天使用前
• 通风口: 每月1次
• 内部灰尘: 每季度1次（需专业人员）

### 6.3 耗材更换指南

🔋 **电池更换指南（如适用）**

⚠️ **警告**: 电池更换必须由专业人员进行

**更换步骤**:
1. 关闭设备并断开所有连接
2. 使用专用工具打开电池仓
3. 断开电池连接器
4. 小心取出旧电池
5. 安装新电池并连接
6. 关闭电池仓并测试功能

**注意事项**:
• 使用原厂或认证电池
• 避免电池短路或损坏
• 正确处理废旧电池
• 更换后进行充放电测试

🔧 **滤网更换指南（如适用）**

**更换周期**: 每3个月或根据使用环境调整

**更换步骤**:
1. 关闭设备并断开电源
2. 打开滤网仓门
3. 轻轻取出旧滤网
4. 清洁滤网仓
5. 安装新滤网（注意方向）
6. 关闭仓门并确认到位
7. 重置滤网计时器（如适用）
