# 农业室内有机肥/室外化肥标准化实施清单

## 1. 农业肥料标准化实施清单概述

### 1.1 标准化体系构成
- **室内有机肥选型层** (权重: 20%): 有机肥类型选择、供应商资质、产品标准符合性
- **室外化肥选型层** (权重: 20%): 化肥类型选择、养分含量、产品标准符合性
- **检测验证层** (权重: 20%): 有机肥腐熟度、卫生安全、养分含量检测；化肥养分含量检测
- **施用操作层** (权重: 20%): 施用方法、施用时间、施用用量控制
- **监测风险控制层** (权重: 20%): 环境监测、风险控制、效果评估

### 1.2 标准化实施目标
- 符合NY/T 525-2021有机肥料标准要求
- 满足GB/T 15063-2020复合肥料标准要求
- 实现NY/T 394-2023绿色食品肥料使用准则
- 确保肥料施用安全、高效、合规

### 1.3 验收标准与阈值
- 总体合规率 ≥ 95%
- 室内有机肥检测项通过率 = 100%
- 室外化肥检测项通过率 = 100%
- 施用操作项通过率 ≥ 95%
- 监测风险控制项通过率 ≥ 90%
