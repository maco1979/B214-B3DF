# 删除浅色主题并优化深色主题样式

- 修改 `ThemeContext.tsx`，将默认主题设置为 'dark'
- 移除主题切换功能，只保留深色主题
- 简化 `useTheme` hook，只返回当前主题

### 1.2 更新全局样式
- 修改 `index.css`，将深色主题变量作为默认值
- 移除 `.dark` 类选择器，直接使用深色主题变量
- 简化 CSS 变量定义，只保留深色主题所需变量

### 1.3 移除主题切换组件
- 移除 `ThemeToggle` 组件在所有页面的使用
