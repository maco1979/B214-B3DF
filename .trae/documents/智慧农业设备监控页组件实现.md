# 智慧农业设备监控页组件实现

## 技术栈
- React + TypeScript + Tailwind CSS
- 符合项目现有的代码风格和组件结构

## 实现步骤

1. **创建新页面组件**：在 `frontend/src/pages/` 目录下创建 `AgricultureMonitoring.tsx` 文件

2. **页面结构设计**：
   - 顶部标题栏：显示"智慧农业设备监控平台"，居中对齐
   - 左侧侧边栏（宽度220px）：包含3个设备列表项
   - 右侧主内容区：分为"实时数据面板"和"设备状态卡片"两个模块

3. **样式实现**：
   - 整体深色模式：背景色#1a1a2e，文字色#f0f0f0
   - 侧边栏项hover时背景色#2d2d44，选中项背景色#3a3a5c
   - 卡片模块：背景色#24243e，圆角8px，内边距20px

4. **交互逻辑**：
   - 使用useState管理选中的设备
   - 点击侧边栏设备项，更新选中状态
   - 右侧"设备状态卡片"根据选中设备显示对应信息

5. **设备数据设计**：
   - 定义设备数据类型和模拟数据
   - 包含设备名称、状态、温度、湿度等信息

6. **组件集成**：
   - 符合项目现有的布局结构
   - 使用项目现有的UI组件和样式规范

## 文件结构

## 预期效果
- 完整的智慧农业设备监控页面
- 响应式设计，适配不同屏幕尺寸
- 流畅的交互体验，点击设备项可切换显示信息
- 符合项目的深色主题设计

## 实现细节
- 使用React Hooks管理状态
- 使用Tailwind CSS实现样式
- 符合项目的代码风格和命名规范
- 包含适当的类型定义
- 提供模拟数据，便于测试和展示
